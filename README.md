# Flights Data Exploratory Analysis

##  Project Overview
This project analyzes flight data using Python, Pandas, and Matplotlib to extract meaningful insights about airline operations, flight patterns, and airport traffic.


##  Objectives

####  Data Analysis
- Understand flight distribution and operating airlines
- Identify busiest airports and popular destinations
- Analyze flight patterns during different time periods


####  Data Cleaning

- Convert date and time formats
- Remove duplicate entries


####  Data Visualization
- Create clear charts to display results
- Compare performance of different airlines
- show air traffic patterns


##  Dataset Information
The dataset contains flight information including:
- Flight numbers and airline details
- Aircraft models and registration
- Airport information (origin and destination)
- Scheduled departure times (UTC and local)
- Flight status and type


##  Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
